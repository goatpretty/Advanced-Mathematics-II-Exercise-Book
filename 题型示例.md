# 题型示例

以下示例均基于 `example_text_type.tex`，展示常见题型的写法与组合方式，方便直接套用。

## 1. 基本填空题
- 用 `qitems` 作为题组环境，单题用 `bbox` 包裹，题干用 `\qitem`，答案空用 `\blankline`。

```latex
\begin{qitems}
    \begin{bbox}
        \qitem 设$f(x)$满足$2f(x)+f(1-x)=x^2 abc$，则
        $\displaystyle\sum \limits_{n}^{0}f(x)=\blankline.$
    \end{bbox}
\end{qitems}
```

## 2. 题干含公式或分段函数
- 题干可直接使用公式与分段函数环境，末尾继续使用 `\blankline`。

```latex
\begin{bbox}
    \qitem 设$f(x)=2x+\sqrt{x^{2}+2x+1}$，$g(x)=$
    \begin{cases}
    x+2, & x\geqslant0, \\
    x-1, & x<0,
    \end{cases}
    则$g[f(x)]=\blankline$.
\end{bbox}
```

## 3. 选择题（四选项）
- 用 `\blankbox` 表示选择题空括号；用 `\fourchoices` 自动排版选项。
- `\textwater` 可插入行内文字水印（内容在 `config.tex` 里配置）。

```latex
\begin{bbox}
    \qitem 可以用\blankbox 定义一个完整的数据结构。\textwater
    \fourchoices{数据元素}{数据对象}{数据关系}{抽象数据类型}
\end{bbox}
```

## 4. 代码题 / 算法题
- 题干中插入 `lstlisting`，可以搭配选择题选项。

```latex
\begin{bbox}
    \qitem 【2017 统考真题】 下列函数的时间复杂度是 \blankbox 。
    \begin{lstlisting}
int func(int n){
    int i=0, sum=0;
    while(sum<n) sum += ++i;
    return i;
}
    \end{lstlisting}
    \fourchoices{$O(\log n)$}{$O(n^{\frac{1}{2}})$}{$O(n)$}{$O(n\log n)$}
\end{bbox}
```

## 5. 含小问的大题
- 使用 `subqitems` + `\subqitem` 表示小问。

```latex
\begin{bbox}
    \qitem 已知曲线$L:y=\ln\sqrt{x}(2\leqslant x\leqslant4)$，记所围成的面积为$S$。
    \begin{subqitems}
        \subqitem 求一点$P_0$，使$S$关于$x$的变化率为零；
        \subqitem 当点$P(x,y)$移动至$(\mathrm{e},\dfrac{1}{2})$时，求$\dfrac{\mathrm{d}S}{\mathrm{d}t}$。
    \end{subqitems}
\end{bbox}
```

## 6. bbox 参数（选择题 / 填空题 / 大题）
- `bbox` 支持可选参数：`[choice]`、`[blank]`、`[big]`。
- 目前用于控制题后间距：`[big]` 追加大题空白；`[blank]` 追加填空题空白；`[choice]` 不加额外空白。

```latex
\begin{bbox}[big]
    \qitem 求过点$A(5,4,3)$且在各坐标轴上的截距相等的平面方程。
\end{bbox}

\begin{bbox}[blank]
    \qitem 方程 $-\dfrac{x^{2}}{4}+\dfrac{y^{2}}{9}=1$ 表示的空间曲面是\blankline。
\end{bbox}

\begin{bbox}[choice]
    \qitem 向量 $\vec{a}$ 与 $\vec{b}$ 的位置关系是\blankbox。
    \fourchoices{平行}{垂直}{相交}{以上都不是}
\end{bbox}
```

## 7. 题内插图
- 用 `\imgin` 插入图片，参数依次为：缩放比例、位置（可空/`l`/`r`）、图片路径。

```latex
\begin{bbox}
    \qitem 以$yOz$面上的平面曲线段$y=f(z)$绕$z$轴旋转一周所成旋转曲面与$xOy$面围成容器，求曲线方程。
    \imgin[0.4]{}{fig/fig01.png}
\end{bbox}
```

## 8. 题组与答案页提示
- 在小节标题后紧接 `\qanswerloc{}`，显示“此部分答案见原书 Pxx”。

```latex
\subsection{文字录入型做题本第一章第1节}\qanswerloc{10}
\begin{qitems}
    % 题目写在这里
\end{qitems}
```
